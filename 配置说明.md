# GitHub Trending → Notion 配置说明

## AI分析功能配置

要启用AI分析功能，需要设置火山引擎豆包API密钥：

### 1. 获取火山引擎 API Key

1. 访问 [火山引擎平台](https://console.volcengine.com/ark)
2. 创建推理接入点，选择豆包模型
3. 获取 API Key 和推理接入点 endpoint

### 2. 设置环境变量

**Windows PowerShell:**
```powershell
# 设置API Key
$env:VOLCANO_API_KEY = "你的API密钥"

# 设置推理接入点（可选，有默认值）
$env:VOLCANO_API_URL = "https://ark.cn-beijing.volces.com/api/v3/chat/completions"

# 设置模型endpoint（必填，从控制台获取）
$env:VOLCANO_MODEL = "ep-xxxxx-xxxxx"

# 永久设置
[System.Environment]::SetEnvironmentVariable('VOLCANO_API_KEY', '你的API密钥', 'User')
[System.Environment]::SetEnvironmentVariable('VOLCANO_MODEL', 'ep-xxxxx-xxxxx', 'User')
```

**Windows CMD:**
```cmd
# 临时设置
set VOLCANO_API_KEY=你的API密钥
set VOLCANO_MODEL=ep-xxxxx-xxxxx

# 永久设置
setx VOLCANO_API_KEY "你的API密钥"
setx VOLCANO_MODEL "ep-xxxxx-xxxxx"
```

**Linux/Mac:**
```bash
# 添加到 ~/.bashrc 或 ~/.zshrc
export VOLCANO_API_KEY="你的API密钥"
export VOLCANO_MODEL="ep-xxxxx-xxxxx"
```

### 3. 运行脚本

设置完成后运行脚本：
```bash
python D:\GitHub-Trending-Notion\github_trending_notion.py
```

## AI输出格式

AI会生成包含以下内容的中文描述：

```
**是什么**：[项目简介]

**有什么用**：[核心功能和用途]

**怎么用**：[安装和使用步骤]
```

## 费用说明

火山引擎豆包API按token计费，具体价格请参考官方定价。

## 其他可选配置

### GitHub Token (可选)

设置后可以提高GitHub API请求限制：

```powershell
$env:GITHUB_TOKEN = "你的GitHub Token"
```
